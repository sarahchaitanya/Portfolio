My Projects

Facial Expression Based Music Using Flask App

Facial expression-based music is an innovative application leveraging Python, OpenCV, and the Eel library to create an interactive and personalized music experience. By analyzing users' facial expressions through real-time video capture, the app can dynamically select and play music that matches the user's current mood.

Developed with the Flask framework, this app integrates computer vision and web technologies to provide a seamless user experience. OpenCV handles the facial expression recognition, while Eel bridges the gap between Python and web-based front-end interfaces, enabling the app to run smoothly in a browser environment. This combination ensures an engaging and responsive music recommendation system based on emotional cues.

Bitcoin Price Prediction Using LSTM

Bitcoin price prediction using LSTM (Long Short-Term Memory) networks is a powerful application of machine learning to forecast cryptocurrency trends. By leveraging historical price data, the LSTM model can identify patterns and make informed predictions about future Bitcoin prices, providing valuable insights for investors.

Developed in Python, this project utilizes libraries such as TensorFlow and Keras for constructing and training the LSTM network. Data preprocessing and visualization are handled by Pandas and Matplotlib, ensuring the accuracy and clarity of the predictive models. This approach offers a sophisticated tool for navigating the volatile cryptocurrency market.
